# Is It Friday?

> *The only question that matters.*

A single-page site that answers the most important question known to mankind. Built with zero frameworks, zero dependencies, and exactly the right amount of effort.

---

## 🚀 Live Demo

[**Find out now →**](https://medivh-dll.github.io/isitfriday)

---

## ✨ Features

- **Tells you if it's Friday.** That's it.
- **If NO** — rotating existential messages that get progressively more devastating
- **If YES** — black confetti rains from the sky. The prophecy is fulfilled.
- **Countdown** — exact days, hours, and minutes until Friday, bottom right corner
- **Check Again button** — it's still no
- Zero dependencies · Zero frameworks · Zero point
